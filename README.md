## 素材

### 图像素材

1. 背景 1920 x 1080
2. 精灵材质 尺寸尽量采用2的倍数。256 512 1024 2048
3. 人物需要面对镜头、背对镜头、朝向左侧和朝向右侧四个方向的贴图。在不考虑手性的前提下，朝左和朝右的贴图可以是镜像关系。
